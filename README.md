# 스프링부트와 AWS로 혼자 구현하는 웹 서비스
## 개요
- springMVC를 통한 팀프로젝트 후 springboot 개념숙지 및 복습을 위한 개인프로젝트  
- AWS 인프라를 통한 자동화 배포
- JUnit4 을 활용한 테스트 코드 작성
- mustache 서버 템플릿 엔진을 통하 화면 구성

## 개발환경
- IDE : IntelliJ IDEA Community
- Git Tools : Source Tree
- OS : Window10 Pro
- SpringBoot 2.1.7
- Java8
- Gradle
- mustache
